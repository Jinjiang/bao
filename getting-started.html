Getting Started

Getting Started

First of all, you need to install Dabao. It's just an one-time shooting.

The Simplest Demo

Create a empty repo and 2 JavaScript files app.js and foo.js.

// app.js
import { foo } from './foo'
foo()

// foo.js
export function foo () {
  console.log('Hello World')
}

export function bar () {
  console.log('Hello World 2')
}

The file app.js is the entry file of this repo. It imports file foo.js.

The whole demo has been here.

Run Dabao

Now we run the command:

$ dabao app.js --output dist/bundle.js

Then you can see a new file dist/bundle.js generated. It's the generated code by Dabao.

The argument --output is to define where to output the generated code (You can also use -o for short).

Hava a Check

Now to have a check, we can run it in Node.js:

$ node dist/bundle.js
> Hello World

Using NPM Scripts

You can also put the command in the toolchain by npm scripts for example:

{
  "name": "dabao-example-simple",
  ...
  "scripts": {
    "build": "dabao app.js -o dist/bundle.js"
  },
  "devDependencies": {
    "dabao": "*"
  }
}

Using NPX

npx is a new way to run npm packages in npm@5.2.0+. You can just run npx dabao without installing Dabao globally:

npx dabao app.js -o dist/bundle.js

That works the same.
