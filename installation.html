Installation

Installation

Global Install

NPM

npm install dabao --global

Yarn

yarn global add dabao

Other Ways

Run with npx Anytime

You can also run dabao with npx which is supported by npm@5.2.0+. See Getting Started

Run with Source Code

If you want to debug dabao, another way to run itself is using link command:

# clone git repo
git clone git@github.com:Jinjiang/dabao.git
cd dabao

# using npm
npm install
npm link

# ... or yarn
yarn
yarn link
